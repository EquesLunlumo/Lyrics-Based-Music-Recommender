# Style Guide
This is a short document describing the preferred coding style for the Groover music app. 

## Languages
Python
JSON
HTML/CSS

## Spaces for Formatting
Python: 4
HTML/CSS: 2

## Naming Conventions
JSON: `camelCase`
Python: `under_scores` unless they are `CONSTANTS`
